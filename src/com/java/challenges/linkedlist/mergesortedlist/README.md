# Merge Sorted Lists

## Problem Statement: ##

**For Example:** In the below two linked list C1 is the intersection node

**Input:** <br />
[<br />
&nbsp;&nbsp;1->4->5,<br />
&nbsp;&nbsp;1->3->4,<br />
&nbsp;&nbsp;2->6<br />
]<br />

**Output:** <br /> 1->1->2->3->4->4->5->6
